1. Introduzione

L'applicazione PMG LiveScore Mobile (“l'App”) è sviluppata da Fabrizio Vacca per la comunità italiana di Pony Mounted Games e Mounted Games. Questa Privacy Policy descrive come l'App gestisce i dati degli utenti nel rispetto del Regolamento UE 2016/679 (GDPR).

3. Dati Raccolti

3.1 Dati NON raccolti

L'App non raccoglie:

  • Dati personali identificativi (nome, email, telefono)
  • Credenziali di accesso (non richiede login/registrazione)
  • Dati di geolocalizzazione
  • Dati di pagamento
  • Contatti, foto o file dal dispositivo

3.2 Dati tecnici di navigazione

L'App effettua richieste HTTPS verso pmglivescore.altervista.org per consultare i dati delle competizioni. Il server di destinazione potrebbe registrare automaticamente dati tecnici (indirizzo IP, user-agent, timestamp). Tali dati sono gestiti esclusivamente dal server di destinazione e non sono raccolti dall'App.

3.3 Cache locale

L'App utilizza un database locale (SQLite) per memorizzare temporaneamente i dati delle competizioni.

  • I dati sono salvati esclusivamente sul dispositivo
  • Non vengono sincronizzati con server esterni
  • Sono eliminati alla disinstallazione dell'App
  • Non sono accessibili da terze parti

4. Finalità del Trattamento

  1. Consultazione risultati: visualizzare dati delle competizioni sportive
  2. Performance: cache locale per ridurre latenza e consumo di banda
  3. Funzionalità offline: accesso ai dati già consultati senza connessione

5. Base Giuridica

  • Legittimo interesse (Art. 6.1.f GDPR): fornire un servizio di consultazione sportiva
  • Consenso implicito: l'utente accetta la policy installando l'App

6. Conservazione dei Dati

  • Cache locale: fino alla disinstallazione dell'App
  • Log server terze parti: secondo le policy del provider (Altervista)

7. Diritti dell'Utente

In conformità al GDPR, l'utente può:

  • Accedere ai dati in cache (visibili nell'App)
  • Cancellare i dati (disinstallando l'App o cancellando la cache)
  • Opporsi al trattamento (cessando l'uso dell'App)

Per esercitare i propri diritti: fabrizio.vacca@gmail.com

8. Sicurezza

  • Connessioni HTTPS per tutte le richieste di rete
  • Isolamento dati locali tramite sandbox del sistema operativo
  • Nessun dato trasmesso a servizi di analytics o tracking
  • Codice sorgente open source e verificabile
