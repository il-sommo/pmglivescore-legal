# Privacy Policy - PMG LiveScore Mobile

**Ultimo aggiornamento**: 8 Marzo 2026

## 1. Introduzione
L'app PMG LiveScore Mobile è progettata per consultare i risultati delle competizioni di Pony Mounted Games. La privacy degli utenti è una nostra priorità.

## 2. Raccolta Dati
L'applicazione **non raccoglie** dati personali identificativi (PII):
- Non è richiesto alcun account o login.
- Non vengono salvati dati di geolocalizzazione.
- Non vengono utilizzati tracker pubblicitari o analytics di terze parti che profilano l'utente.

## 3. Dati Tecnici
L'app effettua richieste HTTP verso `pmglivescore.altervista.org`. Il server di destinazione potrebbe registrare log tecnici standard (indirizzo IP, tipo di dispositivo) necessari per il funzionamento del protocollo HTTP.

## 4. Cache Locale
L'app utilizza un database locale (`sqflite`) per memorizzare temporaneamente i dati degli iscritti e delle gare al fine di migliorare le performance e ridurre il consumo di dati. Questi dati restano esclusivamente sul dispositivo dell'utente.
